# User Roles

React Bricks has 2 user roles:

- **Admin**: they can edit content, [lock/unlock pages](pages#locked-pages), invite users from the [React Bricks Dashboard](/docs/getting-started/react-bricks-dashboard).
- **Editor**: they can edit all the pages, and, based on permissions set by the Admin, the can Add new pages, Delete pages or Trigger a build hook.

## Permissions

**Admin**

- Create pages
- Edit pages' content
- Delete pages
- Trigger a rebuild for static sites (production and staging)
- Lock / Unlock pages
- Invite users to app / Remove them
- Edit App settings

**Editor**

- Edit pages' content
- Create pages _(if allowed by Admin)_
- Delete pages _(if allowed by Admin)_
- Trigger a rebuild for static sites (staging) _(if allowed by Admin)_
- Trigger a rebuild for static sites (production) _(if allowed by Admin)_
